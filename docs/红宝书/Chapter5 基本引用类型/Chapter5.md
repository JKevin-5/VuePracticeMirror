## 5.1 Date

创建日期对象

```js
let now = new Date();
```

日期对象格式化（String字符串转Date对象）

- 月/日/年
- 月名 日，年
- 周几 月名 日 年 时:分:秒 时区

```js
let someDate = new Date(Date.parse("May 23, 2019"));
// 等同于
let someDate = new Date("May 23, 2019");
```
